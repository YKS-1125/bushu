# 🚀 部署操作手册（国内云 · Ubuntu 22.04）

> 服务器买好后，照着这份手册一步步来。全程约 15 分钟。
> 每一段命令都可以**直接复制粘贴**。

---

## 前置：你需要准备的 3 样东西

1. **服务器公网 IP**（腾讯云控制台 → 轻量应用服务器列表里能看到），形如 `123.45.67.89`
2. **服务器登录密码**（买的时候设的，或在控制台「重置密码」）
3. **DeepSeek API 密钥**（在你本地 `backend/.env` 文件里，`DEEPSEEK_API_KEY=` 后面那串）

---

## 第 1 步：开放防火墙端口（在腾讯云控制台网页操作）

进入你的服务器 → **「防火墙」** 标签页 → 添加规则，放行这两个端口：

| 应用类型 | 协议 | 端口 | 说明 |
|---------|------|------|------|
| 自定义   | TCP  | 80   | 网站访问口（必须） |
| SSH     | TCP  | 22   | 远程连接口（一般默认已开） |

> ⚠️ 不开 80 端口，网站建好了外面也访问不到。

---

## 第 2 步：连上服务器

### 方式A（最简单）：网页版终端
腾讯云控制台 → 你的服务器 → 点 **「登录」** 按钮 → 浏览器直接弹出一个黑色终端，已经连好了。**新手推荐这个，不用装任何东西。**

### 方式B：本地 PowerShell 用 SSH
在你自己电脑的 PowerShell 里输入（把 IP 换成你的）：
```
ssh root@123.45.67.89
```
第一次会问 `yes/no`，输 `yes` 回车；然后输入服务器密码（**输密码时屏幕不显示任何字符，是正常的**，输完直接回车）。

---

## 第 3 步：设置 API 密钥（在服务器终端里）

把下面这行的 `你的密钥` 替换成真实密钥，粘贴回车：
```
export DEEPSEEK_API_KEY="你的密钥"
```

---

## 第 4 步：下载并运行一键部署脚本

复制粘贴这一整段，回车：
```
curl -fsSL https://raw.githubusercontent.com/YKS-1125/bushu/main/deploy/deploy.sh -o deploy.sh && bash deploy.sh
```

脚本会自动：装 Docker → 拉代码 → 构建镜像 → 启动网站。
首次构建约 3-5 分钟，耐心等它跑完，最后会打印：
```
🎉 部署完成！
   访问地址： http://你的公网IP
```

---

## 第 5 步：验证上线

浏览器打开 `http://你的公网IP` —— 能看到林小禾网站并正常聊天，就成功了！🎉

把这个网址发给任何人，他们都能访问。

---

## 常用运维命令（收藏备用）

```
docker ps                      # 看网站是否在运行
docker logs -f linxiaohe-ai    # 看实时日志（排查报错）
docker restart linxiaohe-ai    # 重启网站
```

**以后改了代码想更新线上版本**：本地 `git push` 后，在服务器上再跑一次
```
bash deploy.sh
```
它会自动拉最新代码、重新构建、重启。

---

## 常见问题

- **打开网址转圈/打不开**：检查第 1 步的 80 端口有没有放行；`docker ps` 看容器是否在跑。
- **聊天报错**：多半是 API 密钥没设对，`docker logs linxiaohe-ai` 看日志；重设密钥后再跑 `bash deploy.sh`。
- **内存不足/构建卡死**：服务器内存要 ≥ 2GB（向量库吃内存），1GB 会崩。
